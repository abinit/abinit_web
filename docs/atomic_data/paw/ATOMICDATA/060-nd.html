JTH PAW atomic datasets for element Neodynium (Nd)

These atomic data have been carefully tested against all-electron calculations and other available atomic datasets. However, they are provided without warranty. They are recommended for use, but they have to be tested again by users.
Some reference results are given here.

 

    PAW dataset: Nd.GGA_PBE-JTH.xml , Nd.GGA_PBE-JTH.UPF
Status: recommended
Exchange-correlation functional: GGA - PBE
Electronic structure ([core] val): [Kr 4d10] 5s2 5p6 4f3 6s2 5d1
Augmentation radius: 2.51 a.u.
Generator: atompaw-4.2.0.4 (input file)
Core wavefunction file (XML): corewf file)

    PAW dataset: Nd.LDA_PW-JTH.xml , Nd.LDA_PW-JTH.UPF
