Old versions of ABINIT

From the latest version of the ABINIT package, a git clone then git checkout allows one to get all versions starting from 8.1.3 (July 2016 - when ABINIT started to be under git control).

Earlier, ABINIT had been placed under bzr control, and this service has been discontinued. Nevertheless, several ABINIT packages can be downloaded from the ABINIT forge using the links below.

No documentation is available on-line for these old versions, although ample documentation can be found in each package. Github on-line documentation starts with ABINITv9.2.1, see the oldest packages from github ABINIT releases.

  • abinit-7.10.5.tar.gz - 2016 (The version of ABINIT linked to the ABINIT2016 paper)
  • abinit-7.0.5.tar.gz - 2013
  • abinit-5.7.4.tar.gz - 2009 (The version of ABINIT linked to the ABINIT2009 paper)
