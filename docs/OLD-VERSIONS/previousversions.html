Note concerning the use of USPP2Abinit with previous versions of ABINIT: 

  • Version 2.3.0 of USPP2Abinit (and later) is only compatible with ABINIT v5.4.3+.
    You can use it with previous versions of ABINIT provided that you modify the "ABINIT pseudopotential file" after having created it.
    Edit it and replace the "paw4" keyword by "paw3".

  • Version 2.4.0 of USPP2Abinit (and later) is only compatible with ABINIT v6.1.0+,
    if file format is "paw5" (see fourth line of "ABINIT pseudopotential file").
    It is compatible with ABINIT v5.4.3+ if file format is "paw4"
    You can use it with previous versions of ABINIT (produce a "paw4" file) provided that you put "USE_XC_NHAT=0"
    in uspp2abinit.dat file.
