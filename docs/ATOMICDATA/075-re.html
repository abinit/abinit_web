JTH PAW atomic datasets for element Re

 

These atomic data have been carefully tested against all-electron calculations and other available atomic datasets. Some reference results are given here.
They are recommended for use, although they have to be tested again by users...

LDA_PW

[Kr 4d10 4f14] 5s2 5p6 6s2 5d5
Augmentation radius:
rpaw=2.4 a.u.
Generator:
atompaw 4.0.0.8 (input file included in the dataset itself)
