Install Abinit

Installing Abinit follows standard Linux package conventions:

make install

Running make install will place the binary executables in a bin/ subdirectory, the documentation and test files in doc/, and so on.

The installation will occur in the default directory /usr/local/. If you wish to choose a different installation directory, re-run:

configure --prefix=<your chosen install point>

then:

make

and finally:

make install

This allows access to the code for multiple users on a cluster or the installation of several versions of Abinit if necessary.
